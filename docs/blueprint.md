# **App Name**: FieldOps

## Core Features:

- Job Management: Create, update, and manage FTTH job tasks including installation, troubleshooting, and maintenance.
- Role-Based Access Control: Implement access control for Admin, Senior, and Engineer roles, ensuring each user can only access permitted features.
- Form Input Job: Allow engineers to fill in job-related details via a complete and comprehensive input form.
- Geotag Photo Upload: Upload photos of the work site with associated geographic coordinates.
- Telegram Bot Backup: Automatically backup the JSON database to a Telegram bot every 12 hours.
- Admin Dashboard Analytics: Visualize job progress using charts and interactive calendars on the admin dashboard.
- AI Suggestion Tool: Provide AI-driven suggestions to improve the efficiency and effectiveness of field operations; an LLM uses reasoning to determine the utility of the suggestions.

## Style Guidelines:

- Primary color: Indigo (#3F51B5), a professional and calming hue suitable for enterprise applications.
- Background color: Light grayish-blue (#E8EAF6), providing a clean and unobtrusive backdrop.
- Accent color: Teal (#009688) for interactive elements and highlights to draw attention without overwhelming the interface.
- Body and headline font: 'Inter' sans-serif font known for its readability and modern appearance.
- Use Lucide or Heroicons icon pack for consistent and professional visual cues throughout the app.
- Responsive and minimal design using Tailwind CSS with soft shadows and reusable components like Buttons, Cards, and Tables.
- Subtle transitions and feedback animations to enhance user experience and interactivity.
